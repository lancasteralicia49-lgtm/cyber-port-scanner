#  Cyber Port Scanner

Multi-threaded TCP port scanner with a professional GUI built using Python (Tkinter).

---

## Features
- Fast multi-threaded scanning (configurable thread limit)
- Custom IP + port range
- Real-time results table
- Service detection (like Nmap)
- Banner grabbing
- CSV export
- Start / Stop / Clear controls
- Dark-themed UI

---

## Lab Setup
- Kali Linux (scanner)
- Windows 10 VM (target)
- VMware Workstation
